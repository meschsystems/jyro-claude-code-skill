# Jyro-Claude-Code-Skill

A Claude Code skill that provides comprehensive knowledge of the Jyro scripting language, enabling Claude to write correct, performant and idiomatic Jyro scripts.

## Installation

1. **Copy the skill folder** to your project's `.claude/skills/` directory:

   ```
   your-project/
   └── .claude/
       └── skills/
           └── jyro-code/
               └── SKILL.md
   ```

2. Claude Code will automatically detect and use the skill when you work with `.jyro` files or ask about Jyro scripting.

## Usage

Once installed, simply ask Claude Code to help with Jyro tasks:

- "Write a Jyro script that filters users by age"
- "How do I group an array by a field in Jyro?"
- "Fix the syntax error in my .jyro file"

Claude will use the skill's knowledge to provide accurate Jyro code with correct syntax and idioms.

## What's Included

The skill covers:
- Complete language syntax (variables, operators, control flow, functions)
- All built-in functions (arrays, strings, math, objects, utilities)
- Common patterns and best practices
- Debugging strategies
- Important gotchas and limitations
